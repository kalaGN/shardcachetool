# shardcachetool for test

## sync my shard data from mysql to redis

## Installation

Install the latest version with

```bash
$ composer create-project shardcachetool/shardcachetool
```


##  Usage

config the mysqldb and redis info in demo.php

```bash

$ php demo.php

```


### Requirements

- PHP 5.3.10 support.
- tested PHP 5.6.39
